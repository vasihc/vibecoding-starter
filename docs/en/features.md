# Features

## Core Technologies

- **Next.js 13**: Build powerful and scalable React applications with server-side rendering.
- **Firebase**: Leverage Firebase platform for authentication, real-time database, and cloud functions.
- **Tailwind CSS**: Rapidly build custom user interfaces with a utility-first CSS framework.
- **TypeScript**: Type-safe development with full TypeScript support.

## Development Features

- **Automatic Code Splitting**: Optimize performance by splitting JavaScript code into smaller, cacheable chunks.
- **Dynamic Routing**: Create dynamic routes to handle various pages and content.
- **Hot Module Replacement**: Enjoy fast development with hot module replacement for instant code changes.
- **Environment Variables**: Securely manage configuration values using environment variables.
- **ESLint and Prettier**: Maintain code quality and consistency with ESLint and Prettier.

## UI Components

- **shadcn/ui**: Modern, accessible, and customizable component library
- **Tailwind CSS v4**: Latest version of the utility-first CSS framework
- **Dark Mode Support**: Built-in theming capabilities

## Authentication & Security

- Email/Password authentication out of the box
- Protected routes with authentication guards
- Secure environment variable management
- Firebase security rules support

## AI Development Support

- AI-assisted development configuration (AGENTS.md)
- Support for Claude, Cursor, and other AI coding assistants
- Structured prompts and conventions for consistent AI-generated code
