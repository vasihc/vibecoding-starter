# Next.js 13 Firebase Стартовый шаблон

[📄 English Documentation](./README.md)

Современный стартовый шаблон для создания приложений Next.js 13 с Firebase, включающий аутентификацию, базу данных реального времени и поддержку AI-ассистированной разработки.

## 🚀 Быстрый старт

```bash
# Клонировать шаблон
git clone https://github.com/vasihc/vibecoding-starter.git

# Установить зависимости
npm install

# Настроить переменные окружения (см. Настройка Firebase)
cp .env.example .env.local

# Запустить сервер разработки
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) чтобы увидеть ваше приложение.
